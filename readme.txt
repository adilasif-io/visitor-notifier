Sends email notification to admin whenever someone visits the website.

== Description ==

Visitor Notifier is a WordPress plugin that sends email notifications to the website admin whenever a visitor accesses the website. The email notification contains information about the visitor's location, such as country, region, and city, along with the date and time of the visit.

== Installation ==

1. Upload the `visitor-notifier` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Visitor Notifier' settings page to configure email settings.

== Frequently Asked Questions ==

= How do I configure email settings? =

You can configure email settings by navigating to the 'Visitor Notifier' settings page in the WordPress admin area. Here, you can specify the recipient email address, CC email address, and email subject for the notification emails.
